# TechTots International School Website

![TechTots Logo](https://via.placeholder.com/150x50/000080/FFFFFF?text=TechTots)  
*Nurturing young innovators through technology-infused early education*

## 🎯 Project Overview
A modern, responsive website for TechTots International School – Ghana's premier early childhood institution blending Cambridge Curriculum and Reggio Emilia principles with AI-powered learning.

**Color Scheme**:  
- Primary: Navy Blue (`#000080`)  
- Secondary: Sky Blue (`#87CEEB`)  
- Accent: Beige (`#F5F5DC`)  
- Background: White (`#FFFFFF`)  

## ✨ Key Features
- Responsive design for all devices
- Interactive program showcase
- Admissions process workflow
- Virtual campus tour
- Parent dashboard integration
- Gallery with photo/video content
- Contact/application forms
